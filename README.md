тестовая версия бота, которая умеет распознавать марки машин, такие как бмв, мерс, хёндай, лада, порш, тойота 
в качестве класификатора марок используется обученая модель в GTM : https://teachablemachine

будут прекреплены два файла(bot.py(в нем основной код для работы бота) и service.py(в нем функция, которая класифицирует логотип машины))
